# USOCCC 1988

Source: https://www.chessprogramming.org/USOCCC_1988

**[Home](/Main_Page "Main Page") \* [Tournaments](/Tournaments_and_Matches "Tournaments and Matches") \* [United States Open Computer Chess Championship](/United_States_Open_Computer_Chess_Championship "United States Open Computer Chess Championship") \* USOCCC 1988**

**[< Prev](/USOCCC_1987 "USOCCC 1987")**

The **Fourth United States Open Computer Chess Championship** was held in [Mobile, Alabama](https://en.wikipedia.org/wiki/Mobile,_Alabama), April 8-10, [1988](/Timeline#1988 "Timeline") - for the last time. There were four participants playing a double round robin, except the [Fidelity](/Fidelity_Electronics "Fidelity Electronics") entries had byes each other.
