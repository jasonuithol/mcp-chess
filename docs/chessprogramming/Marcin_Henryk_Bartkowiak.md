# Marcin Henryk Bartkowiak

**[Home](/Main_Page "Main Page") \* [People](/People "People") \* Marcin Henryk Bartkowiak**

**Marcin Henryk Bartkowiak**,  
a Polish computer scientist and software engineer at [Ten Square Games](https://en.wikipedia.org/wiki/Ten_Square_Games), holding an [engineer's degree](https://en.wikipedia.org/wiki/Engineer%27s_degree) in [informatics](https://en.wikipedia.org/wiki/Informatics) from [University of Wrocław](https://en.wikipedia.org/wiki/University_of_Wroc%C5%82aw) in 2019 [[1]](#cite_note-1).
He is author of the [UCI](/UCI "UCI") compliant [open source chess engine](/Category:Open_Source "Category:Open Source") [Combusken](/Combusken "Combusken"), written in the [Go programming language](/Go_(Programming_Language) "Go (Programming Language)"), first released in February 2020 [[2]](#cite_note-2).
