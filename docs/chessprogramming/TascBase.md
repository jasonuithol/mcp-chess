**TascBase**,  
a chess database program by [TASC](/TASC "TASC"), developed since 1993 by former [NICBase](/NICBase "NICBase") developers [Hans](/Hans_Kuijf "Hans Kuijf") and [Nico Kuijf](/index.php?title=Nico_Kuijf&action=edit&redlink=1 "Nico Kuijf (page does not exist)"), the [GUI](/GUI "GUI") by [Marc Derksen](/Marc_Derksen "Marc Derksen") was based on the [ChessMachine](/ChessMachine "ChessMachine") GUI. TascBase requires a [80386](/X86 "X86") [PC](/IBM_PC "IBM PC") with [VGA-display](https://en.wikipedia.org/wiki/Video_Graphics_Array) and at least 4 [Mebibyte](https://en.wikipedia.org/wiki/Mebibyte) of [memory](/Memory "Memory"), running as 32-bit program either under [MS-DOS](/MS-DOS "MS-DOS") using the [DOS extender](https://en.wikipedia.org/wiki/DOS_extender) [DOS/4G](https://en.wikipedia.org/wiki/DOS/4G), or [Windows](/Windows "Windows"). TascBase **1.0** was released in October 1994, soon followed by version **1.1**, incorporating [The King](/The_King "The King") engine by [Johan de Koning](/Johan_de_Koning "Johan de Koning") as analysis tool. TascBase **2.0**, released in 1996, features [Ken Thompson's](/Ken_Thompson "Ken Thompson") 5-men [endgame databases](/Thompson%27s_Databases "Thompson's Databases"), followed by TascBase **2.1** in 1997. TascBase was compliant with the [TASC SmartBoard](/TASC_SmartBoard "TASC SmartBoard"), and was also shipped under the name *PerfectBase* without EGTBs [[2]](#cite_note-2).

# Database

The native multi-indexed database structure always connects two consecutive key positions by one chess move. TascBase will find piece and pawn formations, material distribution, or specific combinations. Query results could either represented in a list view of all games in the current key - or as key structure as tree of variations. TascBase supports various formats for import and export, beside its native TascBase format, [ChessBase](/ChessBase_(Database) "ChessBase (Database)") ([CBF](/ChessBase_(Database)#CBF "ChessBase (Database)")), [NICBase](/NICBase "NICBase"), and [PGN](/Portable_Game_Notation "Portable Game Notation") and customizable text formats [[3]](#cite_note-3).
