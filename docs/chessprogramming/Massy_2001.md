## Infinite Loops

[![Pacq elch.jpg](/images/4/49/Pacq_elch.jpg)](http://www.ludochess.com/trn_massy2001/tournoi.php3)

The infinite loop bug of [El Chinito](/Chinito "Chinito") attracts attention, Round 2, [Pacque Expert](/Pacque_Expert "Pacque Expert") - El Chinito [[5]](#cite_note-5)

```
[Event "Massy 2001"]
[Site "Massy"]
[Date "2001.06.10"]
[Round "2"]
[White "Pacque Expert 2"]
[Black "El Chinito"]
[Result "1/2-1/2"]

1.e4 c5 2.Nf3 e6 3.b3 Nc6 4.Bd3 Be7 5.O-O Nf6 6.Bb2 d5 7.exd5 exd5 8.Re1 O-O 
9.Bxf6 Bxf6 10.c3 Qd6 11.Bxh7+ Kxh7 12.d4 cxd4 13.cxd4 Bg4 14.Qd3+ g6 15.Rf1 
Bxf3 16.Qd2 Nxd4 17.h3 Ne2+ 18.Qxe2 Bxe2 19.Nd2 Bxf1 20.Rxf1 Qa3 21.Rd1 Qxa2 
22.Nf3 Qxb3 23.Rd2 Bc3 24.Ng5+ Kg7 25.Rd3 Qb1+ 26.Kh2 Qxd3 27.h4 Be5+ 28.g3 
Rac8 29.Nh3 Rc2 30.Kg2 Qxg3+ 31.Kf1 Qxh3+ 32.Kg1 1/2-1/2
```
