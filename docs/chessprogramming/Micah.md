**Micah**,  
an [UCI](/UCI "UCI") compliant chess engine by [Folkert van Heusden](/Folkert_van_Heusden "Folkert van Heusden"), first released in June 2019 [[1]](#cite_note-1). The engine is available as source [[2]](#cite_note-2), for [Linux](/Linux "Linux"), [Android](/Android "Android") and [Windows](/Windows "Windows") operating systems. Micah is the first chess-program to make use of [libchess](/Libchess "Libchess").
