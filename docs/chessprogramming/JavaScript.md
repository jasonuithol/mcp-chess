# JavaScript

Source: https://www.chessprogramming.org/JavaScript

**[Home](/Main_Page "Main Page") \* [Programming](/Programming "Programming") \* [Languages](/Languages "Languages") \* JavaScript**

**JavaScript** is a dynamically typed programming language with a prototype-based object system. It is interesting because it runs in [web browsers](https://en.wikipedia.org/wiki/Web_browser) and has received more than its fair share of optimization effort. Modern JavaScript engines with [JIT compilation](https://en.wikipedia.org/wiki/Just-in-time_compilation) can play chess well enough to be interesting.
