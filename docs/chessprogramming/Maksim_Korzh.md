Maksym Korzh [[1]](#cite_note-1)

**Maksim (Maksym) Korzh**, [alias](https://en.wikipedia.org/wiki/Alias) Code Monkey King,  
an Ukrainian [freelancer](https://en.wikipedia.org/wiki/Freelancer) and computer chess programmer.
As a computer chess aficionado and chess programming enthusiast, he learned [JavaScript](/JavaScript "JavaScript"), [Python](/Python "Python") and [C](/C "C") languages, as well as [x86](/X86 "X86") [Assembly](/Assembly "Assembly").
He started freelancing at [Upwork](https://en.wikipedia.org/wiki/Upwork) but soon moved to [YouTube](https://en.wikipedia.org/wiki/YouTube) to create
a [channel](https://en.wikipedia.org/wiki/YouTube_(channel)) dedicated to [web scraping](https://en.wikipedia.org/wiki/Web_scraping) and [web development](https://en.wikipedia.org/wiki/Web_development) [[2]](#cite_note-2),
and further combined his profession with his computer chess activity for a YouTube channel dedicated to chess programming - where he introduced chess programming topics along with the development of various chess engines for didactic purpose [[3]](#cite_note-3).
