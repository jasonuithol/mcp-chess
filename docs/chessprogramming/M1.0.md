**m1.0**,  
a chess program by [Maarten van der Meulen](/Maarten_van_der_Meulen "Maarten van der Meulen"), written in [C](/C "C"). It had its tournament debut at the [DOCCC 1984](/DOCCC_1984 "DOCCC 1984"), where it run on a [PDP-11/44](/PDP-11 "PDP-11") [[1]](#cite_note-1), and was already superseded by [Arachne](/Arachne "Arachne") in the following years.
