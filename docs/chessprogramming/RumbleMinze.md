**RumbleMinze**,  
an [UCI](/UCI "UCI") compliant [open source chess engine](/Category:Open_Source "Category:Open Source") by [Erik Madsen](/Erik_Madsen "Erik Madsen"), written in [C#](/C_sharp "C sharp") to run under the [Windows](/Windows "Windows") [.NET framework](https://en.wikipedia.org/wiki/.NET_Framework). It was released in April 2012 [[6]](#cite_note-6), and is predecessor to [MadChess](/MadChess "MadChess") which followed a few months later [[7]](#cite_note-7) introducing a new [move generation](/Move_Generation "Move Generation") technique using the C# [yield statement](https://en.wikipedia.org/wiki/Generator_%28computer_programming%29#C.23) [[8]](#cite_note-8) [[9]](#cite_note-9) [[10]](#cite_note-10) [[11]](#cite_note-11).
