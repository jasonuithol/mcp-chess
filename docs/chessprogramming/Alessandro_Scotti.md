**Alessandro Scotti**,  
an Italian software developer and computer chess programmer. He is author of the chess engines [Kiwi](/Kiwi "Kiwi") [[2]](#cite_note-2)
and [Hamsters](/Hamsters "Hamsters") [[3]](#cite_note-3). Alessandro also tinkered with the [WinBoard](/WinBoard "WinBoard") source code, and was able to add a few new features to it [[4]](#cite_note-4).
