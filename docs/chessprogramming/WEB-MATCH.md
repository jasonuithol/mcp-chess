**WEB-MATCH**, (WEBMATCH)  
a chess game web broadcaster composed by [Ed Schröder](/Ed_Schroder "Ed Schroder"), released in May 2016 [[1]](#cite_note-1). WEB-MATCH works with [tournament managers](/Match_Statistics#TournamentManager "Match Statistics") incorporated in various [interfaces](/GUI "GUI") such as [Arena](/Arena "Arena"), which are able collect all games in one [PGN-file](/Portable_Game_Notation "Portable Game Notation") and to save every played move immediately into that file. WEB-MATCH uses [pgn4web](/index.php?title=Pgn4web&action=edit&redlink=1 "Pgn4web (page does not exist)") with [FTP](https://en.wikipedia.org/wiki/File_Transfer_Protocol) into the root directory of the web-server, and will create the ranking table of all running channels, using [Ordo](/index.php?title=Ordo&action=edit&redlink=1 "Ordo (page does not exist)") to create an Elo list. Based on WEB-MATCH, [Ferdinand Mosca](/Ferdinand_Mosca "Ferdinand Mosca") introduced the feature improved [aiBloodshed](/AiBloodshed "AiBloodshed") in July 2017, incorporating [ChessGUI](/ChessGUI "ChessGUI") playing the games [[2]](#cite_note-2).
