**Quark**,  
a [Chess Engine Communication Protocol](/Chess_Engine_Communication_Protocol "Chess Engine Communication Protocol") compatible chess engine by [Thomas Mayer](/Thomas_Mayer "Thomas Mayer"), written in [C++](/Cpp "Cpp"). Beside using [alpha-beta](/Alpha-Beta "Alpha-Beta"), Quark used to improve about 100 Elo each in implementing a [transposition table](/Transposition_Table "Transposition Table"), [null move pruning](/Null_Move_Pruning "Null Move Pruning") and an advanced [evaluation](/Evaluation "Evaluation") along with various speed improvements [[2]](#cite_note-2).
Over the board, Quark played the [11th World Computer Chess Championship](/WCCC_2003 "WCCC 2003") in Graz 2003 [[3]](#cite_note-3),
as well as various [IPCCCs](/IPCCC "IPCCC") in Paderborn, [Dutch Open Computer Chess Championship](/Dutch_Open_Computer_Chess_Championship "Dutch Open Computer Chess Championship"), [International CSVN Tournament](/International_CSVN_Tournament "International CSVN Tournament") [[4]](#cite_note-4) in Leiden and the [BELCT 2001](/BELCT_2001 "BELCT 2001") [[5]](#cite_note-5) in Berlin.
It further played multiple [CCT Tournaments](/CCT_Tournaments "CCT Tournaments").
