**aiBloodshed**,  
a chess game web broadcaster composed by [Ferdinand Mosca](/Ferdinand_Mosca "Ferdinand Mosca"), released in July 2017 [[1]](#cite_note-1), based on [WEB-MATCH](/WEB-MATCH "WEB-MATCH") by [Ed Schröder](/Ed_Schroder "Ed Schroder") [[2]](#cite_note-2). It further uses [pgn4web](/index.php?title=Pgn4web&action=edit&redlink=1 "Pgn4web (page does not exist)"), [JavaScript](/JavaScript "JavaScript"), [Ordo](/index.php?title=Ordo&action=edit&redlink=1 "Ordo (page does not exist)") [rating system](/index.php?title=Rating_System&action=edit&redlink=1 "Rating System (page does not exist)"), [python-chess](/Python-chess "Python-chess"), [ChessGUI](/ChessGUI "ChessGUI"), [FTP](https://en.wikipedia.org/wiki/File_Transfer_Protocol) and [Highcharts](https://en.wikipedia.org/wiki/Highcharts) to broadcast games with live [Principal Variation](/Principal_Variation "Principal Variation") for [UCI engines](/Category:UCI "Category:UCI").
