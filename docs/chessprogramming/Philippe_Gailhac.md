# Philippe Gailhac

Source: https://www.chessprogramming.org/Philippe_Gailhac

**[Home](/Main_Page "Main Page") \* [People](/People "People") \* Philippe Gailhac**

**Philippe Gailhac**,  
a French software developer and computer [Othello](/Othello "Othello") and computer chess programmer.
He is author of the Othello program [Jacp'Oth](https://www.game-ai-forum.org/icga-tournaments/program.php?id=286),
which won the Bronze medal at the [4th Computer Olympiad 1992](/4th_Computer_Olympiad#Othello "4th Computer Olympiad") in London, and author of the [UCI](/UCI "UCI") compliant chess engine [Philou](/Philou "Philou") [[1]](#cite_note-1).
