**Óscar Toledo Gutiérrez**,  
a Mexican software developer, computer chess programmer and author of the minimalistic chess program [Toledo](/Toledo "Toledo"), which won the 18th [International Obfuscated C Code Contest](https://en.wikipedia.org/wiki/International_Obfuscated_C_Code_Contest) (IOCCC) in 2005, Best Game competition.
With the improved *Toledo Nanochess* , Óscar held the lead in *The World's Smallest Chess Program* [[2]](#cite_note-2), in a tough competition with [H.G. Muller's](/Harm_Geert_Muller "Harm Geert Muller") [Micro-Max](/Micro-Max "Micro-Max").
Óscar further won multiple categories at the 19th IOCCC in 2006 [[3]](#cite_note-3), Most Portable Chess Set for Toledo 2 with [GUI](/GUI "GUI") and Best of Show award with his [Intel](/Intel "Intel") [8080](/8080 "8080") emulator.
