# Dominik Klein

Source: https://www.chessprogramming.org/Dominik_Klein

**[Home](/Main_Page "Main Page") \* [People](/People "People") \* Dominik Klein**

**Dominik Klein**, [[1]](#cite_note-1) [[2]](#cite_note-2)  
a German software developer and as computer chess programmer author of the cross-platform chess [graphical user interface](/GUI "GUI") [Jerry](/Jerry "Jerry") [[3]](#cite_note-3), and the [electronic chess board](/Sensory_Board "Sensory Board") [ArdEBoard](/index.php?title=ArdEBoard&action=edit&redlink=1 "ArdEBoard (page does not exist)") along with [Arduino](/Arduino "Arduino") controller source code [[4]](#cite_note-4).
He is further co-author of the [Kanji](https://en.wikipedia.org/wiki/Kanji) [character recognition](https://en.wikipedia.org/wiki/Optical_character_recognition) software *kanjicanvas* implemented in client side only [JavaScript](/JavaScript "JavaScript") [[5]](#cite_note-5),
and provided a translation of [Yu Nasu's](/Yu_Nasu "Yu Nasu") 2018 paper introducing [NNUE](/NNUE "NNUE") [[6]](#cite_note-6) [[7]](#cite_note-7). In September 2021, Dominik Klein published *Neural Networks For Chess* [[8]](#cite_note-8).
