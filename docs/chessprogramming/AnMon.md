**AnMon**,  
a chess program by [Christian Barreteau](/Christian_Barreteau "Christian Barreteau"), one of the first engines which supported [WinBoard](/WinBoard "WinBoard") aka the [Chess Engine Communication Protocol](/Chess_Engine_Communication_Protocol "Chess Engine Communication Protocol"), later also [UCI](/UCI "UCI"). AnMon was available as [Young Talent](/ChessBase#YoungTalents "ChessBase") by [ChessBase](/ChessBase "ChessBase") running under the [Fritz6 GUI](/Fritz#FritzGUI "Fritz"), and is [Arena](/Arena "Arena") partner engine [[3]](#cite_note-3) . It is a [tactically](/Tactics "Tactics") very strong [MTD(f)](/MTD(f) "MTD(f)") searcher.

AnMon participated at various [French Computer Chess Championships](/French_Computer_Chess_Championship "French Computer Chess Championship") and had its debut at the [FCCC 1995](/FCCC_1995 "FCCC 1995"), further the [WMCCC 1997](/WMCCC_1997 "WMCCC 1997") in [Paris](https://en.wikipedia.org/wiki/Paris), and multiple [Massy](https://en.wikipedia.org/wiki/Massy%2C_Essonne) [French Programmers Tournaments](/French_Programmers_Tournament "French Programmers Tournament"), winning [Massy 2000](/index.php?title=Massy_2000&action=edit&redlink=1 "Massy 2000 (page does not exist)"), [Massy 2005](/Massy_2005 "Massy 2005") and [Massy 2006](/Massy_2006 "Massy 2006").

# Description

given in 1997 from the [ICGA](/ICGA "ICGA") site [[5]](#cite_note-5) :

```
The program AnMon (half ANgel, half deMON) is written in C and run under Windows NT. It was developed as a hobby for 8 years. AnMon participated in the 1995, and 1996 French Microcomputer Chess Championship. It is based on MTD(f) and uses null-move pruning, selective extensions, transposition tables, killer heuristic, and quiescence search. AnMon is a tactical program.
```

```
AnMon uses an opening book containing about 100.000 positions. It was established by the program. AnMon search about 50.000 nodes per second on a Pentium 2 at 233 MHz.
```
