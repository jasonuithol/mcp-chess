**Crux**, (Crux Chess for Windows)  
a chess engine by [László Szalai](/L%C3%A1szl%C3%B3_Szalai "László Szalai"), written in [Delphi](/Delphi "Delphi"). **Crux Chess for Windows** runs under [Windows](/Windows "Windows") either in console mode or with an own [GUI](/GUI "GUI"), the [WinBoard](/WinBoard "WinBoard") version [[2]](#cite_note-2) supports the [Chess Engine Communication Protocol](/Chess_Engine_Communication_Protocol "Chess Engine Communication Protocol").
The search is plain [alpha-beta](/Alpha-Beta "Alpha-Beta") improved by [null move pruning](/Null_Move_Pruning "Null Move Pruning") and the [killer heuristic](/Killer_Heuristic "Killer Heuristic").
Crux played the [ICT 2001](/ICT_2001 "ICT 2001") and various [Hungarian Chessprogram Open](/Hungarian_Chessprogram_Open "Hungarian Chessprogram Open"), where it won the [MASPV 2002](/MASPV_2002 "MASPV 2002"). Crux' [opening book](/Opening_Book "Opening Book") is edited by [Pandix](/Pandix "Pandix") [book author](/Category:Opening_Book_Author "Category:Opening Book Author") [Csaba Szûts](/Csaba_Sz%C3%BBts "Csaba Szûts") [[3]](#cite_note-3).
