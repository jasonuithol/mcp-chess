**python-chess**,  
a chess software [library](https://en.wikipedia.org/wiki/Library_(computing)) created by [Niklas Fiekas](/Niklas_Fiekas "Niklas Fiekas"), written in [Python](/Python "Python") and released under the [GPL v3](/Free_Software_Foundation#GPL "Free Software Foundation"). The goal was to create a simple and relatively highlevel library [[1]](#cite_note-1). It supports Python 3.5.3+ and [PyPy3](https://en.wikipedia.org/wiki/PyPy) [[2]](#cite_note-2).
