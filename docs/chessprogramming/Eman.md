# Eman

Source: https://www.chessprogramming.org/Eman

**[Home](/Main_Page "Main Page") \* [Engines](/Engines "Engines") \* [Stockfish](/Stockfish "Stockfish") \* Eman**

**Eman**,  
a closed source [Stockfish derivative](/Stockfish#Derivatives "Stockfish") by [Khalid Omar](/Khalid_Omar "Khalid Omar"), affiliated with [InfinityChess](/index.php?title=InfinityChess&action=edit&redlink=1 "InfinityChess (page does not exist)") based in [Abu Dhabi](https://en.wikipedia.org/wiki/Abu_Dhabi), [United Arab Emirates](https://en.wikipedia.org/wiki/United_Arab_Emirates).
Using the allegedly private engine requires registering at InfinityChess, downloading their client application, and sending a [private message](https://en.wikipedia.org/wiki/Personal_message) to Eman's author at InfinityChess Forum
[[1]](#cite_note-1)
[[2]](#cite_note-2).
Khalid Omar may then provide a link to download and use the engine - prior to Eman 6.60 along with a personalized, optional time limited license key, checked via InfinityChess server while starting the engine [[3]](#cite_note-3).
Supporting this kind of a [freemium](https://en.wikipedia.org/wiki/Freemium) business model in ignorance of possible [GPL](/Free_Software_Foundation#GPL "Free Software Foundation") [violations](https://en.wikipedia.org/wiki/Gpl-violations.org),
Eman is received with enthusiasm inside InfinityChess' target group, in particular by certain [correspondence chess](https://en.wikipedia.org/wiki/Correspondence_chess) and [freestyle](https://en.wikipedia.org/wiki/Advanced_Chess) players.
Eman regularly include latest Stockfish updates, like [NNUE](/NNUE "NNUE"), and emphasizes own implementations concerning [book learning](/Book_Learning "Book Learning") and [persistent hash table](/Persistent_Hash_Table "Persistent Hash Table"), [dynamic contempt](/Contempt_Factor "Contempt Factor"),
and a feature dubbed *coherence evaluation*, addressing score equalities by picking up different material and positional components [[4]](#cite_note-4).
