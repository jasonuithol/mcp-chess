# Igor Korshunov

Source: https://www.chessprogramming.org/Igor_Korshunov

**[Home](/Main_Page "Main Page") \* [People](/People "People") \* Igor Korshunov**

**Igor V. Korshunov**,  
a Belorussian chess programmer, and author of the chess program [WildCat](/WildCat "WildCat") [[1]](#cite_note-1), which won the [1st Computer Chess Championship of CIS Countries](/CCCCISC_2008 "CCCCISC 2008") in 2008, and the [Russian draughts](/Checkers "Checkers") program *Kallisto* [[2]](#cite_note-2) [[3]](#cite_note-3).
Along with [Jury Osipov](/Jury_Osipov "Jury Osipov"), Igor Korshunov is co-author of the chess program [Belka](/Belka "Belka") [[4]](#cite_note-4). In May 2010, Igor Korshunov introduced his new [open source engine](/Category:Open_Source "Category:Open Source") [Murka](/Murka "Murka") [[5]](#cite_note-5), in July 2013 available as version 3.0 [[6]](#cite_note-6).
Igor is moderator of the [Kasparov computer chess forum](/Computer_Chess_Forums "Computer Chess Forums") [[7]](#cite_note-7).
