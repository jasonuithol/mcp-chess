**Tenberg Basic Chess**, (Chess in Basic)  
an early chess program written in [Basic](/Basic "Basic"), published in 1977 by *Tenberg Software Systems*, [Cockeysville, Maryland](https://en.wikipedia.org/wiki/Cockeysville,_Maryland),
sold on [paper tape](https://en.wikipedia.org/wiki/Punched_tape) with documentation for $20 [[1]](#cite_note-1).
The program takes 16K bytes of [memory](/Memory "Memory") - for [teletypes](https://en.wikipedia.org/wiki/Teletype_Model_33) ans [CRT's](https://en.wikipedia.org/wiki/Cathode-ray_tube), and played the [Second West Coast Computer Faire Microcomputer Chess Tournament](/MCCT_1978 "MCCT 1978") in March 1978 [[2]](#cite_note-2).
