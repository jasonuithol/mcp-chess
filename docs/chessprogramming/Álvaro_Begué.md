# Álvaro Begué

Source: https://www.chessprogramming.org/Álvaro_Begué

**[Home](/Main_Page "Main Page") \* [People](/People "People") \* Álvaro Begué**

**Álvaro Begué**,  
a Spanish [computer checkers](/Checkers "Checkers"), [computer go](/Go "Go") and computer chess programmer with expertise in [machine learning](/Learning "Learning") and [automated tuning](/Automated_Tuning "Automated Tuning"), along with [José Manuel Morán](/Jos%C3%A9_Manuel_Mor%C3%A1n "José Manuel Morán") co-author of the Spanish Checkers [[1]](#cite_note-1) program *Torquemada* [[2]](#cite_note-2) [[3]](#cite_note-3) and the chess program [Ruy Lopez](/Ruy_Lopez "Ruy Lopez"). Along with [John Tromp](/John_Tromp "John Tromp"), he is co-author of the Go playing program *Dimwit* [[4]](#cite_note-4).

Since 2013, Álvaro is in the process to write a new chess engine [[5]](#cite_note-5). In November 2016, he published the open source [tuning](/Automated_Tuning "Automated Tuning") code [RuyTune](/RuyTune "RuyTune"), an instance of a [logistic regression](/Automated_Tuning#LogisticRegression "Automated Tuning") performing [limited-memory BFGS](https://en.wikipedia.org/wiki/Limited-memory_BFGS) along with [reverse-mode automatic differentiation](https://en.wikipedia.org/wiki/Automatic_differentiation#Reverse_accumulation) [[6]](#cite_note-6) [[7]](#cite_note-7), and in June 2017 he published the [open source engine](/Category:Open_Source "Category:Open Source") [RuyDos](/RuyDos "RuyDos") [[8]](#cite_note-8), co-authored by José Manuel Morán [[9]](#cite_note-9).
