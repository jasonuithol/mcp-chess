# Ron Murawski

Source: https://www.chessprogramming.org/Ron_Murawski

**[Home](/Main_Page "Main Page") \* [People](/People "People") \* Ron Murawski**

**Ron Murawski**,  
an American computer chess programmer, and primary author of the [Chess Engine Communication Protocol](/Chess_Engine_Communication_Protocol "Chess Engine Communication Protocol") compatible chess engine [Horizon](/Horizon "Horizon") [[1]](#cite_note-1).
He already had first chess programming experience in the 80s, where he wrote a chess [move generator](/Move_Generation "Move Generation") in [Z80](/Z80 "Z80") [assembly](/Assembly "Assembly"), running [perft](/Perft "Perft") on the [Sinclair ZX81](/Sinclair_ZX81 "Sinclair ZX81"). After that he did not return to chess programming until 2000 [[2]](#cite_note-2).
Ron hosts and maintains the **Computer-Chess Wiki** [[3]](#cite_note-3), providing information on [engines](/Engines "Engines") including engine downloads [[4]](#cite_note-4), [engine protocols](/Protocols "Protocols"), and [chess GUIs](/GUI "GUI"), how to play tournaments and more.
