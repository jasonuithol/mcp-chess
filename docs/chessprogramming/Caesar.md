Caesar [[1]](#cite_note-1)

**Caesar**,   
an early microcomputer chess program written by [John Lowe](/index.php?title=John_Lowe&action=edit&redlink=1 "John Lowe (page does not exist)"), apparently in [Z80](/Z80 "Z80") [assembly](/Assembly "Assembly"). Caesar played the [2nd European Microcomputer Chess Championship](/European_MCC_1981 "European MCC 1981") in 1981, and the [4th European Microcomputer Chess Championship](/European_MCC_1983 "European MCC 1983") 1983 in [London](https://en.wikipedia.org/wiki/London), running on a [TRS-80](/TRS-80 "TRS-80") [[2]](#cite_note-2).
