Judd Niemann [[1]](#cite_note-1)

**Judd Niemann**,  
an Australian software engineer, IT professional, musician and jazz guitarist. Interested in computer chess, he is author [JuddChess](/index.php?title=JuddChess&action=edit&redlink=1 "JuddChess (page does not exist)"), written in [C++ 11](/Cpp "Cpp") [[2]](#cite_note-2)
and the [move generation](/Move_Generation "Move Generation") engine from JuddChess, the [quad-bitboard](/Quad-Bitboards "Quad-Bitboards") based [Perft](/Perft "Perft") utility [Juddperft](/index.php?title=Juddperft&action=edit&redlink=1 "Juddperft (page does not exist)") [[3]](#cite_note-3),
which - translated from C++ to [C#](/C_sharp "C sharp") - was used by [David J. Elliott](/index.php?title=David_J._Elliott&action=edit&redlink=1 "David J. Elliott (page does not exist)") to improve the [Ceres](/Ceres "Ceres") [Lc0](/Leela_Chess_Zero#Lc0 "Leela Chess Zero") frontend [[4]](#cite_note-4) [[5]](#cite_note-5).
