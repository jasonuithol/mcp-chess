# Vladimir Medvedev

Source: https://www.chessprogramming.org/Vladimir_Medvedev

**[Home](/Main_Page "Main Page") \* [People](/People "People") \* Vladimir Medvedev**

**Vladimir R. Medvedev**,  
a Russian games and chess programmer, and author of the chess program [GreKo](/GreKo "GreKo"), first released on April 01, 2002 [[1]](#cite_note-1). However, Vladimir already started some chess programming in 2000 [[2]](#cite_note-2), with the aim to write a standalone chess program with an own [GUI](/GUI "GUI") using [MFC](https://en.wikipedia.org/wiki/Microsoft_Foundation_Class_Library), but implemented an [Othello](/Othello "Othello") applet in [Java](/Java "Java") first [[3]](#cite_note-3). On one of his download sites, Vladimir does not only provide GreKo, but also a [Kaissa](/Kaissa "Kaissa") [PC](/IBM_PC "IBM PC") port in [Turbo C](/C "C") from 1992, and a listing of the [ITEP Chess Program](/ITEP_Chess_Program "ITEP Chess Program") for the [M-20](/M-20 "M-20") computer [[4]](#cite_note-4) [[5]](#cite_note-5).
Vladimir further published [machine learning](/Learning "Learning") related articles on determining [point values](/Point_Value "Point Value") by [logistic regression](/Automated_Tuning#LogisticRegression "Automated Tuning") [[6]](#cite_note-6) and [Texel's Tuning Method](/Texel%27s_Tuning_Method "Texel's Tuning Method") as applied to [GreKo](/GreKo "GreKo") [[7]](#cite_note-7) in the popular Russian IT portal [Habrahabr](https://en.wikipedia.org/wiki/Habrahabr).
