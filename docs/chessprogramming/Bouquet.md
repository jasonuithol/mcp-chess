[Lubin Baugin](/Category:Lubin_Baugin "Category:Lubin Baugin") - Still-life with Chessboard [[1]](#cite_note-1)

**Bouquet**,  
an [UCI](/UCI "UCI") compliant [Ippolit](/Ippolit "Ippolit") [derivative](/Category:Derivative "Category:Derivative") served as closed source by [Jose Mº Velasco](/Jose_Maria_Velasco "Jose Maria Velasco") [[2]](#cite_note-2).
Bouquet was first released in May 2012 [[3]](#cite_note-3),
and required [MSVC++ 2010](https://en.wikipedia.org/wiki/Visual_C%2B%2B) [dlls](https://en.wikipedia.org/wiki/Dynamic-link_library) [[4]](#cite_note-4).
32- and 64-bit executables for respective [Windows](/Windows "Windows") versions were released - currently no longer available.
Bouquet 1.8 has a [parallel search](/Parallel_Search "Parallel Search") which supports up to 16 [threads](/Thread "Thread"), and uses runtime-detection whether the [x86](/X86 "X86") or [x86-64](/X86-64 "X86-64") [population count](/Population_Count "Population Count") instruction is available or not [[5]](#cite_note-5) .
