**Ivan Bacigal**,  
an American computer chess programmer and author of the chess program [Waxman](/Waxman "Waxman"), first released in 1991 as as [MS-DOS](/MS-DOS "MS-DOS") engine, in 1992 as 16-bit [Windows](/Windows "Windows") program, and in 2008 as [WinBoard](/WinBoard "WinBoard") compliant engine [[1]](#cite_note-1), latest version from 2017 so far [[2]](#cite_note-2).
