**libchess**,  
a software [library](https://en.wikipedia.org/wiki/Library_(computing)) to allow a developer to quickly implement a chess program. It is build-up from [C++17](/index.php?title=C%2B%2B&action=edit&redlink=1 "C++ (page does not exist)") header-files and contains all code for [move generation](/Move_Generation "Move Generation") and [position](/Chess_Position "Chess Position") [tracking](/Incremental_Updates "Incremental Updates"). [Evaluation](/Evaluation "Evaluation") and [search](/Search "Search") are kept for the developer. Libchess is developed by [Manik Charan](/Manik_Charan "Manik Charan") and can be retrieved from [GitHub](https://en.wikipedia.org/wiki/GitHub) [[1]](#cite_note-1).
