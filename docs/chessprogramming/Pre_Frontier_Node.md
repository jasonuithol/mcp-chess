# Pre Frontier Node

Source: https://www.chessprogramming.org/Pre_Frontier_Node

**[Home](/Main_Page "Main Page") \* [Search](/Search "Search") \* [Node](/Node "Node") \* Pre Frontier Node**

**Pre Frontier nodes** are [nodes](/Node "Node") immediately above [Frontier Nodes](/Frontier_Nodes "Frontier Nodes"), typically being searched at [depth](/Depth "Depth") of 2. This is where [extended futility pruning](/Futility_Pruning#Extendedfutilityprunning "Futility Pruning") might be applied. The definition is taken from the papers of [Ernst A. Heinz](/Ernst_A._Heinz "Ernst A. Heinz") [[1]](#cite_note-1). [Robert Hyatt](/Robert_Hyatt "Robert Hyatt") defines pre frontier nodes as depth == 1 nodes [[2]](#cite_note-2).
