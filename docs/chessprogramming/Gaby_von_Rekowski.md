**Gaby von Rekowski**
a female German chess programmer. Along with [Reinhold Gellner](/Reinhold_Gellner "Reinhold Gellner"), Gaby von Rekowski co-authored the chess program [Nightmare](/Nightmare_GER "Nightmare GER") [[1]](#cite_note-1).
