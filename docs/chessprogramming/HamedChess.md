HamedChess [[1]](#cite_note-1)

**HamedChess**,  
a [Windows](/Windows "Windows") chess program by [Hamed Ahmadi](/Hamed_Ahmadi "Hamed Ahmadi") which comes with an own [GUI](/GUI "GUI").
HamedChess was written in [C++](/Cpp "Cpp") in 2002/2003 using [Microsoft Visual Studio](https://en.wikipedia.org/wiki/Microsoft_Visual_Studio) [[2]](#cite_note-2).
It applies [alpha-beta](/Alpha-Beta "Alpha-Beta") along with a few optimizations and improvements to allow it to search deeper [[3]](#cite_note-3).
