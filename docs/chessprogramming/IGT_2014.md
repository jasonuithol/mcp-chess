# Selected Games

[CyberPagno](/CyberPagno "CyberPagno") was the dedicated tournament winner with a half point ahead of [Pedone](/Pedone "Pedone") before the last round, and already announced mate in 9 versus [AdaChess](/AdaChess "AdaChess").
Apparently due to a bug, instead of the winning 48...Nd4, CyberPagno played the disasterous 48...d4 to the same [target square](/Target_Square "Target Square") and lost the game and the title [[3]](#cite_note-3).
