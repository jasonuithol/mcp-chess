[GNU Chess 5's](/GNU_Chess "GNU Chess") precursor program by [Chua Kong Sian](/Chua_Kong_Sian "Chua Kong Sian"), which maintained a separate but cooperative parallel development path [[2]](#cite_note-2).
The GNU Chess 5 Header comment mention GNU Chess is based on the two research programs **Cobalt** by Chua Kong Sian and [Gazebo](/Gazebo "Gazebo") by [Stuart Cracraft](/Stuart_Cracraft "Stuart Cracraft") [[3]](#cite_note-3).
