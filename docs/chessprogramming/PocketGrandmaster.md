**PocketGrandmaster**,  
a chess program for [Windows](/Windows "Windows") [Pocket PCs](/index.php?title=Pocket_PC&action=edit&redlink=1 "Pocket PC (page does not exist)") and [mobiles](https://en.wikipedia.org/wiki/Windows_Mobile) by [Frank Schneider](/Frank_Schneider "Frank Schneider") and [Kai Skibbe](/Kai_Skibbe "Kai Skibbe"), initially based on [GromitChess 3.10.3](/Gromit "Gromit") [[2]](#cite_note-2), successor of GromitChess 3.9.5 which finished 4th at [WMCCC 2001](/WMCCC_2001 "WMCCC 2001") winning the title of the World Microcomputer Amateur Chess Champion [[3]](#cite_note-3), and subsequently, by Gromit's successor [Anaconda](/Anaconda "Anaconda").
PocketGrandmaster is [shareware](https://en.wikipedia.org/wiki/Shareware) and some features are limited in the unregistered version [[4]](#cite_note-4) [[5]](#cite_note-5).
PocketGrandmaster features an own [GUI](/GUI "GUI") with [UCI](/UCI "UCI") like protocol [[6]](#cite_note-6) with additional adapted chess engines available for [ARM](https://en.wikipedia.org/wiki/ARM_architecture) and [XScale](https://en.wikipedia.org/wiki/XScale) processor devices [[7]](#cite_note-7). PocketGrandmaster supports the [DGT Board](/DGT_Board "DGT Board") and can act as [FICS](/index.php?title=Free_Internet_Chess_Server&action=edit&redlink=1 "Free Internet Chess Server (page does not exist)") and [ICS](/index.php?title=Internet_Chess_Club&action=edit&redlink=1 "Internet Chess Club (page does not exist)") client.
