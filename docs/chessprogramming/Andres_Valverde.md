Andres Valverde [[1]](#cite_note-1)

**Andres Valverde Toresano**,  
a Spanish [agricultural engineer](https://en.wikipedia.org/wiki/Agricultural_engineering), chess player and chess programmer.
He has written the [correspondence or email chess](https://en.wikipedia.org/wiki/Correspondence_chess) tool *ECTool* [[2]](#cite_note-2),
a client program for *XfccBasic* [[3]](#cite_note-3) [Web Services](https://en.wikipedia.org/wiki/Web_service),
the [Chess Engine Communication Protocol](/Chess_Engine_Communication_Protocol "Chess Engine Communication Protocol") compatible chess engine [EveAnn](/EveAnn "EveAnn"), and in collaboration with [Pradu Kannan](/Pradu_Kannan "Pradu Kannan") and [Fonzy Bluemers](/Fonzy_Bluemers "Fonzy Bluemers") the [WinBoard](/WinBoard "WinBoard") engine [Dirty](/Dirty "Dirty"), which is a hybrid between [Buzz](/Buzz "Buzz") ([Search](/Search "Search")) and EveAnn ([Evaluation](/Evaluation "Evaluation")) [[4]](#cite_note-4).
