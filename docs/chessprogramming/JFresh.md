# JFresh

Source: https://www.chessprogramming.org/JFresh

**[Home](/Main_Page "Main Page") \* [Engines](/Engines "Engines") \* JFresh**

**JFresh**,  
an [UCI](/UCI "UCI") compliant [open source chess engine](/Category:Open_Source "Category:Open Source") by [Christian Daley](/Christian_Daley "Christian Daley"), written in [C](/C "C") and distributed under the [GNU General Public License](/Free_Software_Foundation#GPL "Free Software Foundation"), first released in March 2014 [[1]](#cite_note-1).
In 2016, JFresh was superseded by [Spartan](/Spartan "Spartan") [[2]](#cite_note-2) [[3]](#cite_note-3).
