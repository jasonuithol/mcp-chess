# Youri Matiounine

Source: https://www.chessprogramming.org/Youri_Matiounine

**[Home](/Main_Page "Main Page") \* [People](/People "People") \* Youri Matiounine**

**Youri Matiounine**,  
an American [actuary](https://en.wikipedia.org/wiki/Actuary) at [New York Life Insurance Company](https://en.wikipedia.org/wiki/New_York_Life_Insurance_Company) [[1]](#cite_note-1). As computer games programmer, Youri Matiounine is author of the [Othello](/Othello "Othello") bot *Ymatioun* [[2]](#cite_note-2), open source under the [GNU GPL v3](/Free_Software_Foundation#GPL "Free Software Foundation") [[3]](#cite_note-3), written in [C++](/Cpp "Cpp") using [AVX2](/AVX2 "AVX2") intrinsics and some [x86-64](/X86-64 "X86-64") [assembly](/Assembly "Assembly") with AVX2 instructions, and the [Chess Engine Communication Protocol](/Chess_Engine_Communication_Protocol "Chess Engine Communication Protocol") and later [UCI](/UCI "UCI") compliant chess engine [Fizbo](/Fizbo "Fizbo"), introduced in May 2014 to [CCRL](/CCRL "CCRL") [[4]](#cite_note-4), which became [open source](/Category:Open_Source "Category:Open Source") in October 2019 [[5]](#cite_note-5) [[6]](#cite_note-6).
