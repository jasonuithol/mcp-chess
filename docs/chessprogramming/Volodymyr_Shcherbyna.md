**Volodymyr Shcherbyna**,  
an Ukrainian software developer and computer chess programmer.
He is author of the [UCI](/UCI "UCI") compliant [open source chess engine](/Category:Open_Source "Category:Open Source") [Igel](/Igel "Igel") [[1]](#cite_note-1), which recently features its own [NNUE](/NNUE "NNUE") implementation [[2]](#cite_note-2).
