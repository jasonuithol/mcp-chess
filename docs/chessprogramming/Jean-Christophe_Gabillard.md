Jean-Christophe Gabillard [[1]](#cite_note-1)

**Jean-Christophe Gabillard**,  
a French computer scientist and IT manager [[2]](#cite_note-2),
chess player [[3]](#cite_note-3),
and as computer chess programmer author of the [Chess Engine Communication Protocol](/Chess_Engine_Communication_Protocol "Chess Engine Communication Protocol") and [UCI](/UCI "UCI") compatible chess engine [Nejmet](/Nejmet "Nejmet") [[4]](#cite_note-4).
