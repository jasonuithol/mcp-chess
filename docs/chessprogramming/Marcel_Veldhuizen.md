Marcel Veldhuizen [[1]](#cite_note-1)

**Marcel Veldhuizen**,  
a Dutch software architect and professional software developer, focused mainly on [Microsoft](/Microsoft "Microsoft") [.NET](https://en.wikipedia.org/wiki/.NET_Framework) development [[2]](#cite_note-2).
Despite not considered as co-author of [Johan Hutting's](/index.php?title=Johan_Hutting&action=edit&redlink=1 "Johan Hutting (page does not exist)") chess engine [31337/Celes](/Celes "Celes"),
he contributed to the engine by working on the interface code and various speedup tips [[3]](#cite_note-3)
