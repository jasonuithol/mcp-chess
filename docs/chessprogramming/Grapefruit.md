Pink Grapefruit [[1]](#cite_note-1)

**Grapefruit**,  
an [UCI](/UCI "UCI") compliant, [derivative](/Category:Derivative "Category:Derivative") [open source chess engine](/Category:Open_Source "Category:Open Source") by [ThinkingALot](/ThinkingALot "ThinkingALot") based on [Toga II 1.4 beta5c](/Toga#1.4beta5c "Toga") [[2]](#cite_note-2), which already had a [parallel search](/Parallel_Search "Parallel Search").
Grapefruit is written in [C++](/Cpp "Cpp") and licensed under the [GPL v2.0](/Free_Software_Foundation#GPL "Free Software Foundation").
