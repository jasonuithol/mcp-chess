**Hironori Ishibashi**, ([Ishibashi](https://en.wikipedia.org/wiki/Ishibashi_(surname)) is surname, 石橋宏之, alias MetalPhaeton [[1]](#cite_note-1) [[2]](#cite_note-2)  
a Japanese software developer, computer chess programmer, and author of the [UCI](/UCI "UCI") compliant [open source chess engine](/Category:Open_Source "Category:Open Source") [Sayuri](/Sayuri "Sayuri") [[3]](#cite_note-3).
