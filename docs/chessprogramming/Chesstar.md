**Chesstar**,  
a chess program written by [Lincoln Shannon](/index.php?title=Lincoln_Shannon&action=edit&redlink=1 "Lincoln Shannon (page does not exist)") [[1]](#cite_note-1) which played the [ACM 1976](/ACM_1976 "ACM 1976"), running on a [Honeywell x16](https://en.wikipedia.org/wiki/Honeywell_316) 16-bit minicomputer [[2]](#cite_note-2).
